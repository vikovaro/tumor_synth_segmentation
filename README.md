# Система автоматизированной сегментации опухолей головного мозга на МРТ

## Описание проекта
Разрабатываемая система предназначена для автоматизированной сегментации опухолей головного мозга на МРТ-снимках с использованием методов машинного обучения. Система интегрирует инструменты обработки данных, классификации МРТ-последовательностей и нейросетевой сегментации для повышения точности и скорости анализа медицинских изображений.

## Используемые технологии и инструменты
- Orthanc: сервер для хранения и обмена DICOM-изображениями.
- DICOMweb API: интерфейс для работы с медицинскими изображениями.
- PostgreSQL: база данных для хранения метаданных и результатов.
-  nnU-Net: нейросетевая архитектура для сегментации медицинских изображений.
- TumorSynth: инструмент на основе сверточных нейронных сетей для сегментации опухолей.
- FreeSurfer: набор инструментов для анализа данных нейровизуализации.
- NIfTI: формат для хранения многомерных медицинских изображений.
